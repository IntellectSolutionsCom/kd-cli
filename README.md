# kd — issue tracker

Public issue tracker for **kd** (Knowledge Dispatcher) — a command-line and Python tool for accessing and managing information that lives in issue trackers and knowledge bases, designed to be equally natural for humans and large language models.

- **Install**: `pip install kd-cli` (or `pipx install kd-cli`)
- **PyPI**: https://pypi.org/project/kd-cli/
- **Report bugs / request features**: [open an Issue](https://github.com/IntellectSolutionsCom/kd-cli/issues/new)

The canonical code repository is private. Patches are welcome via issue-based discussion; a minimal public code mirror may be added if demand appears.

## Supported backends

- **YouTrack** (today)
- **JIRA, GitHub Issues, GitLab, Obsidian** — on the roadmap

## Before filing an issue

Please include:

- `kd --version`
- Your OS and Python version
- The exact command you ran and the observed output
- What you expected instead

For security-sensitive reports, email the maintainer rather than filing a public issue.
